# ☕ Cafe Management System

A full-stack **Cafe Management System** developed using **Python (Flask)** that helps manage daily cafe operations efficiently. This project is designed to handle orders, billing, menu management, and customer records in a simple and user-friendly way.

## 🚀 Features

* 🧾 Order Management (Add / Update / Delete Orders)
* 💳 Billing System with total calculation
* 📋 Menu Management (Add new items, update prices)
* 👥 Customer Details Management
* 📊 Simple and clean UI for easy navigation
* ⚡ Fast and responsive system

## 🛠️ Tech Stack

* Backend: Python (Flask)
* Frontend: HTML, CSS, Bootstrap
* Database: SQLite / MySQL

## ⚙️ Installation

1. Clone the repository

   ```bash
   git clone https://github.com/your-username/cafe-management-system.git
   ```
2. Navigate to the project folder

   ```bash
   cd cafe-management-system
   ```
3. Install dependencies

   ```bash
   pip install -r requirements.txt
   ```
4. Run the application

   ```bash
   python app.py
   ```
